# Westernacher Brand Visual Standards

This file defines the brand visual identity for Westernacher PPT slides. When generating slides, ALWAYS prepend these specifications to the image generation prompt.

## Color Rules (严格执行)

### Primary Brand Colors (必须使用)
| Element | Color | Hex | Rule |
|---------|-------|-----|------|
| 章节标题 (Section Title) | Dark Gray | #3C4652 | 必须使用深灰色，不可用其他颜色 |
| 主标题 (Action Title) | Blue | #1596D1 | 必须使用蓝色，不可用其他颜色 |
| 正文文字 | Dark Gray | #3C4652 | 所有正文内容使用深灰色 |
| 页面背景 | White | #FFFFFF | 白色背景，不可使用其他底色 |
| 强调/高亮 | Blue | #1596D1 | 关键数据、重点标注使用蓝色 |

### Secondary Colors (辅助使用)

灰色系（用于次要文字、分隔线、图表辅助）：
- #282E36 深色背景块
- #5D6A76 次要文字、分隔线
- #858F98 图表辅助色、弱化元素

蓝色系（用于数据可视化、背景色块）：
- #72B7E2 图表、数据可视化
- #92CCEA 浅色背景块、渐变

紫色系（用于复杂图形、插图）：
- #3E276D 深紫
- #7768AE 中紫
- #CECAE5 浅紫

棕色系（用于复杂图形、暖色调）：
- #8B744D 深棕
- #BDA06E 中棕
- #D8C6A8 浅棕

### Color Prohibitions (颜色禁止)
- 禁止使用品牌色板以外的任何颜色
- 禁止在图片中出现颜色色卡、色板展示、颜色样本
- 禁止将颜色的 hex 值作为文字显示在幻灯片上
- 标题颜色不可互换：章节标题必须深灰色，主标题必须蓝色

## Typography Rules (排版规则)

- 字体：微软雅黑 (Microsoft YaHei)
- 章节标题：深灰色 #3C4652，10pt，页面左上角
- 主标题 (Action Title)：蓝色 #1596D1，16-18pt，加粗，章节标题下方
- 正文：深灰色 #3C4652，10-12pt
- 数据标注/KPI：蓝色 #1596D1，可加大加粗
- 页码：右下角，8pt，灰色 #858F98

## Icons

- Style: Line Icons (线性图标), clean and minimal
- Weight: Consistent thin stroke
- Color: Match the section color (Dark Gray for neutral, Blue for accent)

## Output Restrictions (严格禁止出现)

The following MUST NOT appear in the generated image:
- 颜色色卡、色板、颜色样本展示
- Hex 颜色值文字（如 #3C4652）
- 百分比标注（如 "30%", "50%"）
- 字体大小标签（如 "24pt", "12pt"）
- Section markers（如 "section", "top", "bottom"）
- 布局描述或元标注
- Westernacher logo
- 任何品牌规范以外的颜色
