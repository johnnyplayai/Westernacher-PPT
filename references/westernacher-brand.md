# Westernacher Brand Visual Standards

This file defines the brand visual identity for Westernacher PPT slides. When generating slides, ALWAYS prepend these specifications to the image generation prompt.

## Primary Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Dark Gray | #3C4652 | Primary text, headers, dark backgrounds |
| White | #FFFFFF | Page background, reverse text |
| Blue | #1596D1 | Accent, action titles, key highlights |



## Secondary Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Mid Gray | #5D6A76 | Secondary text, dividers |
| Light Gray | #858F98 | Chart accents, subtle elements |
| Mid Blue | #72B7E2 | Data visualization, charts |
| Light Blue | #92CCEA | Background blocks, gradients |



## Icons

- Style: Line Icons (线性图标), clean and minimal
- Weight: Consistent thin stroke
- Color: Match the section color (Dark Gray for neutral, Blue for accent)



## Output Restrictions

The following MUST NOT appear in the generated image:
- Percentage annotations (e.g., "30%", "50%")
- Font size labels (e.g., "24pt", "12pt")
- Section markers (e.g., "section", "top", "bottom")
- Layout descriptors or meta-annotations
- Westernacher logo
